# Korean style Poker game (섯다)

### 프로젝트 계획 및 설명
> 2019년 05월 기준
> nodejs와 socket.io를 이용하여 섯다 게임을 구현
> nodejs와 javascript의 기본을 익히고 가장 처음 구현한 웹
> 최대 4명이서 게임 플레이 가능

### Deploy by 

https://koreans-style-poker.herokuapp.com/
